# React Work

This repository serves as a base for various React applications, each organized in its own folder.

## Folder Structure

The repository is organized with each app residing in its own folder:

```bash
react-work/
|-- flashcards/
|   |-- src/
|   |   |-- components/
|   |   |   |-- App.jsx
|   |   |-- data/
|   |   |   |-- data.js
|   |   |-- App.css
|   |   |-- index.html
|   |   |-- main.js
|-- get-advice/
|   |-- src/
|   |   |-- components/
|   |   |   |-- App.jsx
|   |   |-- App.css
|   |   |-- index.html
|   |   |-- main.js
|-- ...
```
